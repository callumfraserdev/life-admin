# Life Admin – Bills & Deadlines Tracker

## Problem
People forget bills, renewals, insurance, MOTs, rent changes.
This causes stress, late fees, and financial damage.

## Goal
A simple tool that ensures you never forget a bill or renewal again. 

Stay simple

Solve one core pain extremely well

Resist feature creep

## Value Proposition
“Never miss a bill, renewal, or deadline again.” 

## MVP (Phase 1)
- Add bill name
- Add due date
- Get reminders

Nothing more.

## Non-goals (for now)
- Budgeting
- Bank integrations
- Expense tracking
- Fancy analytics

## Monetisation
- Freemium with limits
- £3–£7/month
- Target: 500 users × £6 = £3,000/month

## Why This Matters
This is a real problem experienced by almost everyone.
The goal is simplicity, clarity, and reliability.

## Status
Planning / Learning phase.
